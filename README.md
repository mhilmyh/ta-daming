# Data Mining Project

We make SIR model with python from [COVID-19 Italy dataset](https://github.com/pcm-dpc/COVID-19).

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install required packages.

```bash
pip install numpy
pip install matplotlib
```
